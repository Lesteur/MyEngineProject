# MyEngineProject

## Introduction

C'est un projet C++ qui a pour but de recréer un moteur d'application avec des ressources entièrement statiques.

Il possède une base d'abstraction qui lui permet de s'adapter à diverses plateformes.

C'est le moteur idéal pour créer des programmes sur des systèmes embarqués ou pour recréer des jeux de consoles rétro !
